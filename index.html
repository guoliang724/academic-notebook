    📥 导入 AI 解析语料 (JSON 格式)

    支持两种格式：单条语料 {...} 或批量数组 [{...}, {...}]

    🔁 兼容字段别名：translation_tips → insights ｜ grammar 字符串自动转换 ｜ takeaway 追加至心法

    字段名 类型 含义 / 对应页面位置
    title string 语料标题，显示在左侧导航栏列表中（建议格式："序号. 主题"）
    genre string 体裁 / 学科领域标签，显示在原文卡右上角的小徽章（如 "法律英语 / 合同文本"）
    body string 英文原文语料段落，显示在顶部蓝色发光原文卡中（斜体大字）
    translation string 精雕中文译文，显示在「译文对比 & 技巧」Tab 的第一张卡片中
    insights string[] 翻译心法列表，每个字符串是一条独立技巧，显示为列表项（同一 Tab 第二张卡）
    grammar object[] 语法剖析条目数组，显示在「语法句式剖析」Tab。每条对象含三个字段：
    title — 语法现象名称（加粗）
    skeleton — 句型骨架拆解（蓝色等宽字体）
    notes — 详细分析说明（灰色小字）
    vocab object[] 词汇表数组，显示在「重点词汇」Tab 的表格。每条对象含四个字段：
    word — 词汇或短语
    type — 词性（n. / v. / adj. 等）
    meaning — 中文释义
    rating — 学术推荐度（如 ⭐⭐⭐⭐⭐）
    specialHTML string? 【可选】自定义 HTML 片段，支持 MathJax 数学公式渲染，显示在词汇板块底部的蓝色卡片中
